This is an iPhone app using the icndb API.

Upon launch it will download 50 random Chuck Norris jokes using the API and sorts them by 
length before displaying them to the user in an interactive Table.

Features:
- The user can scroll through the jokes from shortest to longest.
- The user can also sort the joke by their category.
- The user can try to get more jokes from a category or simply get more random jokes.
- If the user has enable their phone with a Facebook and/or Twitter profile form the 
  settings page, the user can click on a joke and then choose to Post or Tweet the Joke.
